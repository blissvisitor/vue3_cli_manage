# 组件打包发布

组件发布时，版本号的定制需遵循前端开发复用规范中的版本章节，以下示例的打包目录为`dist`

## 打包

运行打包命名 `npm run pkg` 完成组件的打包,运行命令后将结果打包到dist文件夹,下图展示了打包后的目录

![img](../../images/depoly/001.png)
